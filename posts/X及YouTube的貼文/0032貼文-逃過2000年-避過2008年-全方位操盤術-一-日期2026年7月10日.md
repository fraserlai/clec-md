**過去三十五年** ( 1980 - 2016 ) **市場有兩次空頭** ( 2000 **、** 2008 ) **，但是經濟學家與華爾街預測了十次以上。投資者應該了解，市場大部份時間是上漲，真正須要賣股票的時機很少，但卻是關鍵。**

**避免虧損有兩個關鍵** :（這是為我過去的歷史做個回憶紀錄，現在：有錢就買打死不賣，絕對不建議操作。） **一、買進時機**! **二、賣出時機**!

**今天談賣出時機[** **關鍵** **]，我是如何在** 2000**年** **和** 2008**年** **避開空頭** ( BEAR MARKET ) **風險。**

**在** 2000**年我還在** IT **產業，出差到美國，在旅館的早餐上，聽到電視新聞播報某某大公司財報不佳。從**1990**年到**2000**年，每年市場絕大部份的公司財報都是高於市場預期，突然那天早上聽到財報不佳，我以為我聽錯了，結果不是，財報真的令市場失望。**

**我的商業知識與在市場經驗的直覺，告訴我出事了，我早餐也不吃了，開會遲到也不管了，立即回到房間，賣掉所有持股。之後，才去客戶那邊開會。就這樣我避開了**2000**年的網路泡沫。直到**2003 **年觸底。**

( **我在**2004**年退休，專職用自己的錢投資與從事投資教學。** )
