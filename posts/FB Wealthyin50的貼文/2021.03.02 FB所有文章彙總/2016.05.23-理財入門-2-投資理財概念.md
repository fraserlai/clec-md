投資理財概念 [理財入門(2)]

年輕人剛踏入社會上班之後都會有一份固定薪水，而薪水除了用來支付每月固定開銷及部份急預備金之外，如果還有剩餘的錢 就是閒置資金了。

大部分的人會將這些閒置資金存入銀行裡來換取極少的利息收入 ( 由其近年來少於2% ) ；有些人則購置股票、債券.....等金融商品 來提高閒置資金的價值。 因此 因應投資人多元化的需求，就有了各式各樣投資產品的誕生。

投資理財就是將閒置資金進行合理安排 並投入高報酬的投資，使所賺的錢能夠日益增進, 因而讓財富創造財富，用賺錢到的錢再去賺錢。

一個人若想致富，所面臨的挑戰不僅在於如何獲得財富，更重要的是如何利用財富創造財富。因此，[ 投資理財規劃 ]是投資人生最初期的理財概念。有良好的理財規劃 便可以創造一個未來富裕的人生。

財投資正確學習心態:

接下來, 我們要先談談 投資人的 [ 理財投資正確學習心態 ]

什麼才是正確的投資學習心態？ 第一件是就是先 ” 拋棄 ” 你以前對投資的認知和操作方式, 忘記所有人 ( 包括專家們 ) 告訴你如何 ” 殺進殺出 ” 的方法, 必須 ” 放空 ” 自己，承認自己對短線的市場震盪一無所知。還有很重要的 – 要改變 以往在股市裡 ” 賺錢的習慣 ”, 重新學習正確的投資。

[ 學習正確的投資 ] 在投資人生裡 是非常重要的一課 。沒有正確的投資觀 是無法達成財富自由的。

詳細 [ 學習正確理財投資 ] 的步驟 會在 [ 理財入門 ]章節裡一一的解說。
